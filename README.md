Philippine Tax Calculator
A web-based tax calculator built with PHP that computes income tax and mandatory government contributions based on the Philippine TRAIN Law and current SSS, PhilHealth, and Pag-IBIG contribution tables.
Features

Automatic tax calculation based on TRAIN Law progressive tax rates
Mandatory contributions computation (SSS, PhilHealth, Pag-IBIG)
Multiple salary frequency options (Daily, Weekly, Bi-monthly, Monthly, Annually)
Real-time calculations with detailed breakdown
Annual projections and effective tax rate display
Clean, responsive user interface

Prerequisites

XAMPP (or any local PHP server)
Web browser
PHP 7.0 or higher

Installation & Setup

Clone or download this repository
Place the project folder in your XAMPP htdocs directory
Start Apache server in XAMPP Control Panel
Open your browser and navigate to http://localhost/tax-calculator

How to Use

Enter your gross salary amount
Select your salary frequency from the dropdown menu (Daily, Weekly, Bi-monthly, Monthly, or Annually)
(Optional) Add any other deductions
The calculator will automatically process and display:

Tax breakdown with applicable bracket
Mandatory government contributions (employee and employer shares)
Net take-home pay
Annual projections and effective tax rate



Language used

PHP - Server-side logic and calculations
HTML/CSS - User interface structure and styling
JavaScript - Client-side interactions and output formatting
